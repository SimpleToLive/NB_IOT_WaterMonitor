模块说明：
	BC26模块			->使用 串口AT指令
	PH模块				->使用 ADC 	OUT:0~3.3V
	TSW-30模块			->使用 ADC	OUT:0~4.5V
	电压线性装换模块	->IN:0~5V	OUT:0~3.3V
	DS18B20				->使用单总线

连线说明：
	BC26 				连接串口3							PB10(T) PB11(R)
	PH					连接ADC1 通道1 即PA1				PA1
	TSW-30				连接电压线性转换模块输入端			
	电压线性转换模块	连接ADC1 通道2 即PA2				PA2
	DS18B20				连接PA11							PA11
